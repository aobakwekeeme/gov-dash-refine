# Spaza Shop Registration & Management System (SSRMS)

A comprehensive digital platform for managing spaza shop registration, compliance, and quality assurance across South Africa.

## Features

- **Multi-role Authentication**: Customer, Shop Owner, and Government Official portals
- **Shop Registration & Management**: Complete business registration workflow
- **Document Management**: Upload and verify compliance documents
- **Inspection System**: Schedule and complete inspections with scoring
- **Security**: Auto-logout, input validation, password management
- **South African Standards**: Phone/name validation conforming to SA rules

## Tech Stack

- React 18 + TypeScript + Vite
- Tailwind CSS
- Supabase (Auth, Database, Storage)
- React Router

## Quick Start

```bash
npm install
npm run dev
```

See inline documentation for detailed information.
